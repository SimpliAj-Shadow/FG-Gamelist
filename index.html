SteamBuddy Help & Features

Welcome to SteamBuddy! This extension enhances your Steam experience by automating redirects, checking game availability from repack sites, and displaying review summaries. Below, you'll find explanations for all features.

üöÄ Open in Steam Client (Redirect)

Automatically redirects Steam Store links (e.g., from web searches or external sites) to open directly in the Steam client using the steam://store/[APPID] protocol.

  • Works on game pages like https://store.steampowered.com/app/12345/.
  • Handles PatchBot redirects (e.g., affiliate links).
  • Toggle: Enable/disable via the main switch in the popup.
  • Delay: 2-second buffer to ensure page loads before redirect.

Tip: If Steam isn't running, it will launch the client.

üõ°Ô∏è Auto Age Verification

Bypasses Steam's age gate on mature-rated games by auto-filling a default birthdate (18+ years ago) and submitting the form.

  • Triggers on pages like https://store.steampowered.com/agecheck/app/[APPID]/.
  • Sets: Day=1, Month=January, Year=Current Year - 18.
  • Always on when redirect is enabled.

Note: This is for convenience; ensure you're of legal age to view content.

‚≠ê Ratings Badge (Steam Reviews)

Displays a purple badge below the game title showing Steam user review summary.

  • Fetches recent reviews via Steam API.
  • Shows: Positive percentage, score description (e.g., "Very Positive"), and total review count.
  • Example: "Steam-Reviews: 85% Very Positive (12,345 Reviews)".
  • Toggle: Enable/disable in popup; reloads Steam tabs on change.

Fallback: "Not available" if data can't be loaded.

‚úÖ FitGirl Repack Availability

Scans for game availability on FitGirl Repacks site and shows colored badges under the game title.

  • Green badge: Exact match found (with download link and optional magnet).
  • Yellow badge: Similar title detected (check manually).
  • Red badge: Not available.
  • Toggle: Enable/disable in popup; reloads Steam tabs on change.
  • Uses fuzzy matching (Levenshtein distance) for accuracy.

Data: Fetched from GitHub repo on startup/install.

üéÅ Check for Free Games & DLCs

One-click button in popup to open Steam's free games/search page with filters for free items and DLCs.

  • URL: https://store.steampowered.com/search?maxprice=free&specials=1&ndl=1.
  • Opens in new tab.

Troubleshooting

  • Redirects not working? Ensure Steam client is installed and the main toggle is ON.
  • Badges not showing? Ensure the relevant toggles (Steam Reviews, FitGirl, or Dodi) are ON and reload Steam tabs via the popup.
  • Lists empty? Extension fetches on startup; check network in console (F12).
  • Update: Click toggles to reload lists/tabs.
